## Setting Up Your Wallet

To participate in the CKB Community Fund DAO, you'll need a compatible wallet. This guide will walk you through the setup process.

### Supported Wallets

The following wallets are currently supported:

- **JoyID** (Recommended)
- **Neuron Wallet**
- **Portal Wallet**

### Step-by-Step Guide

<Steps>

<Step>

### Install Your Wallet

Download and install your preferred wallet from the official website.

</Step>

<Step>

### Create or Import Account

Follow the wallet's instructions to create a new account or import an existing one.

</Step>

<Step>

### Connect to DAO Platform

Visit the DAO platform and click "Connect Wallet" to link your wallet.

</Step>

</Steps>
