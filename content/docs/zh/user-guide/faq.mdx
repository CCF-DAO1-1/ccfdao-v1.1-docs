## 钱包与连接

<Accordions>
<Accordion title="支持哪些钱包？">
DAO 平台基于 [CCC](https://github.com/ckb-devrel/ccc)（Common Chains Connector）构建，支持以下钱包类型：

- **EVM 钱包**：MetaMask、imToken、CoinBase、OKX Wallet 等所有 EIP-6963 兼容钱包
- **Bitcoin 钱包**：UniSat、Xverse、OKX Wallet、UTXO Global
- **CKB 原生钱包**：JoyID、REI Wallet、UTXO Global Wallet
- **Nostr 钱包**：所有 NIP-07 兼容钱包
</Accordion>

<Accordion title="推荐使用哪个钱包？">
如果你是新用户，我们推荐使用 **JoyID** 钱包。JoyID 是一个多链钱包，无需下载任何应用，直接在浏览器中访问 [app.joy.id](https://app.joy.id) 即可创建和使用。
</Accordion>

<Accordion title="为什么我的钱包连接失败？">
请检查以下几点：

1. 确保你的钱包扩展如 MetaMask 已安装并解锁
2. 确保你的浏览器允许钱包扩展访问当前页面
3. 尝试刷新页面后重新连接
4. 如果使用 JoyID，请确保你的浏览器支持 WebAuthn
</Accordion>
</Accordions>

## 投票权与参与资格

<Accordions>
<Accordion title="我想参与治理，需要做哪些准备？">
要参与 DAO 治理投票，你需要满足以下条件：

1. **连接 CKB 钱包** - 使用支持的钱包连接到 DAO 平台
2. **创建 Web5 DID** - 在平台上创建你的去中心化身份
3. **存入 Nervos DAO** - 你的 CKB 钱包地址（也即你创建 Web5 DID 的钱包地址）需要在 [NervDAO](https://nervdao.com) 中有存款，或者你在 DAO 治理平台的个人中心绑定了 Neuron 钱包里的存款地址。绑定教程参见👉：[绑定 NervosDAO 地址](/zh/docs/user-guide/getting-started/bind-nervosdao-address)。
</Accordion>

<Accordion title="我使用 Neuron 钱包存入了 Nervos DAO，可以投票吗？">
可以。你可以将 Neuron 钱包里锁定了 NervosDAO 的钱包地址绑定到你在 DAO 平台连接的 Web5 DID，这样同样可以获得投票权。具体的绑定过程，请参考👉：[绑定 NervosDAO 地址](/zh/docs/user-guide/getting-started/bind-nervosdao-address)。
</Accordion>

<Accordion title="投票权重是如何计算的？">
投票权重基于你在 Nervos DAO 中的存款金额。存入的 CKB 越多，投票权重越大。一个 Web5 DID 对应一个 Owner 地址，每个 Owner 地址可以绑定多个 CKB 钱包地址。对于某个 Web5 DID，其投票权重等于其 Owner 地址本身的NervosDAO 余额 + 与之绑定的所有 CKB 钱包地址的 NervosDAO 存款总额。

举例说明：
- Alice 使用 JoyID 钱包地址 A 注册了一个 Web5 DID，名为 alice，同时该钱包地址 A 在 nervdao.com 上存了 100,000 CKB 在 NervosDAO里。
- 此外，Alice 还在 Neuron 的钱包地址 B 上存了 500,000 CKB。
- Alice 通过 DAO 平台将地址 B 与其 Web5 DID 绑定。
这样 alice 的总投票权重为：
 100,000 CKB + 500,000 CKB = 600,000 CKB。

具体计算规则请参考[投票机制](/zh/docs/meta-rule/voting-mechanism)文档。
</Accordion>

<Accordion title="什么是 Web5 DID？为什么需要它？">
Web5 DID（去中心化身份）是一种基于区块链的身份标识。在 DAO 平台中，DID 用于：

- 唯一标识投票者身份
- 防止重复投票
- 关联你的钱包地址与投票记录
</Accordion>

<Accordion title="谁可以投票？">
DAO v1.1 的投票通过智能合约实现。在提案发起投票时，系统会收集当前所有已注册的 Web5 DID 账号对应的 Owner 地址（CKB 地址）作为**投票白名单**。只有在白名单内的 Web5 DID 账号才能参与该提案的投票。

社区成员可以将自己的 Nervos DAO 地址绑定到白名单地址中，以增加投票权重。
</Accordion>

<Accordion title="我刚注册了 Web5 DID，为什么无法参与当前投票？">
这是因为投票白名单是在**提案发起投票时**生成的。如果你在某个提案已经进入投票阶段后才注册 Web5 DID，由于发起投票时系统没有收集到你的 Web5 DID，所以你无法参与该提案的投票。

但你可以参与之后发起的新提案投票，只要在新提案发起投票之前完成 Web5 DID 注册即可。
</Accordion>

<Accordion title="如何确保我能参与投票？">
为确保你能参与投票，建议：

1. **尽早注册 Web5 DID** - 在你感兴趣的提案进入投票阶段之前完成注册
2. **关注提案进度** - 留意社区评议期的提案，在其进入投票阶段前完成准备
3. **绑定 Nervos DAO 地址** - 将你存有 CKB 的 Nervos DAO 地址绑定到 Web5 DID，以获得投票权重
</Accordion>
</Accordions>

## 提案与投票

<Accordions>
<Accordion title="如何提交提案？">
提案需要经过以下流程：

1. 在 DAO 平台发起提案，积极参与社区讨论并收集反馈
2. 根据反馈完善提案内容
3. 满足社区评议期要求后，提案人可以选择是否进入立项投票阶段
4. 进入立项投票阶段，等待社区投票结果

<Callout title="注意" type="warning">
一旦进入立项投票阶段，意味着提案已正式进入决策流程。
</Callout>


详细流程请参考[提案生命周期](/zh/docs/meta-rule/proposal-lifecycle)文档。
</Accordion>

<Accordion title="投票期限是多久？">
投票期限根据投票类型不同：

| 投票类型 | 投票时间 |
|---------|---------|
| 立项投票 | 7 天 |
| 快速确认投票（里程碑拨款） | 3 天 |
| 复核投票 | 7 天 |
| 最终裁决投票 | 7 天 |

详情请参考[投票机制](/zh/docs/meta-rule/voting-mechanism)文档。
</Accordion>

<Accordion title="投票后可以更改吗？">
在投票期结束前，你可以更改你的投票。一旦投票期结束，投票结果将被锁定，无法更改。
</Accordion>

<Accordion title="提案通过的条件是什么？">
提案通过需要满足：

1. **法定人数**：参与投票的权重达到最低门槛
2. **支持率**：支持票占比超过规定比例

不同类型的提案可能有不同的通过条件，详情请参考[投票机制](/zh/docs/meta-rule/voting-mechanism)文档。
</Accordion>
</Accordions>

## CKB 获取

<Accordions>
<Accordion title="如何获取 CKB？">
CKB 可通过[主流 CEX 交易所](https://coinmarketcap.com/currencies/nervos-network/#Markets)购买。
</Accordion>

<Accordion title="需要多少 CKB 才能参与投票？">
没有最低 CKB 数量要求，但你需要在 Nervos DAO 中有存款才能获得投票权。存款金额会影响你的投票权重。
</Accordion>
</Accordions>

## 其他问题

<Accordions>
<Accordion title="DAO 物业是什么？">
DAO 物业（DAO Stewards）是负责 DAO 日常运营和维护的团队。他们的职责包括：

- 维护 DAO 基础设施
- 协调社区讨论
- 监督提案执行
- 处理紧急事务

详情请参考 [DAO 物业](/zh/docs/overview/dao-stewards)文档。
</Accordion>
