## 为 DAO v1.1 做贡献

本节为希望了解架构并为 CKB 社区基金 DAO v1.1 代码库做出贡献的开发者提供全面的技术文档。

### 涵盖内容

- **架构**：深入了解技术设计、智能合约和链下服务
- **开发环境**：如何设置本地环境和运行测试
- **贡献指南**：代码风格、PR 流程和功能开发的最佳实践

## 前置要求

在开始开发之前，你应该具备：

- 区块链概念的基本理解
- 熟悉 CKB 和 Nervos 网络
- TypeScript/JavaScript 经验
- 智能合约开发知识（用于合约贡献）
